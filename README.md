# AI Challenge (Python)

Единый репозиторий для многонедельного обучения по AI.

## Структура

- `weeks/` — задания и материалы по неделям.
- `src/common/` — общий код, который можно переиспользовать между неделями.
- `pyproject.toml` — базовая конфигурация Python-проекта.

## Как начать

1. Убедиться, что установлен Python 3.11+.
2. Создать и активировать виртуальное окружение:
   - `python3 -m venv .venv`
   - `source .venv/bin/activate`
3. Обновить pip:
   - `python -m pip install --upgrade pip`

## Рабочий процесс по неделям

Для каждой новой недели добавляй папку вида:

- `weeks/week01/`
- `weeks/week02/`
- `weeks/week03/`

Рекомендуемый шаблон внутри недели:

- `README.md` — цель недели и чеклист.
- `task_01_*.py`, `task_02_*.py` — файлы с заданиями.
- `data/` — локальные данные (если нужны).
- `notes.md` — заметки и выводы.
