# Week 02

## Цель

Реализовать первого агента как отдельную сущность с простым web-интерфейсом на Flask.

## Чеклист

- [ ] Поднять и активировать `.venv`
- [ ] Установить зависимости проекта (`pip install -e .`)
- [ ] Запустить `task1/task_01_flask_agent.py`
- [ ] Проверить, что агент принимает запросы из UI
- [ ] Проверить, что агент вызывает YandexGPT API и возвращает ответ
- [ ] Проверить, что в UI хранится скользящая история последних 5 сообщений
- [ ] Поэкспериментировать с `system prompt` в UI
- [ ] Зафиксировать вывод и заметки

## Файлы недели

- `task1/agent.py` — инкапсулированная логика агента и вызова API.
- `task1/task_01_flask_agent.py` — Flask-приложение (роуты и хранение истории).
- `task1/templates/index.html` — простой чат-интерфейс.

## Запуск

```bash
python3 weeks/week02/task1/task_01_flask_agent.py
```

После запуска откройте:

- http://127.0.0.1:5000

## Требования к окружению

Должны быть заданы переменные:

- `YANDEX_API_KEY`
- `YANDEX_FOLDER_ID`
- `YANDEX_MODEL` (опционально, по умолчанию используется `yandexgpt-lite`)
